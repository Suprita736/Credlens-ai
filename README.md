# CredLens AI — Phase 1 MVP

CredLens AI is a lightweight Chrome extension designed to detect potentially misleading factual claims in YouTube Shorts using the Gemini 1.5 Flash API.

## Features (Phase 1)
- **YouTube Shorts Detection**: Automatically detects when you are watching a Short.
- **Swipe Ignore System**: Skips processing for videos watched for less than 3 seconds to save API calls.
- **Real-time Caption Extraction**: Scrapes active subtitles for analysis.
- **Gemini 1.5 Flash Integration**: Identifies factual claims and filters out satire/comedy.
- **Minimal Shadow DOM Overlay**: Shows a non-intrusive credibility indicator.

## Tech Stack
- **Frontend**: React, Tailwind CSS, TypeScript
- **AI**: Gemini 1.5 Flash API
- **Extension**: Manifest V3, Content Scripts, Service Worker
- **Build**: Vite

## Setup Instructions

### 1. Prerequisites
- Node.js installed.
- A Gemini API Key from [Google AI Studio](https://aistudio.google.com/).

### 2. Installation
1. Clone the repository or extract the files.
2. Run `npm install` to install dependencies.
3. Run `npm run build` to generate the `dist` folder.

### 3. Load Extension in Chrome
1. Open Chrome and go to `chrome://extensions/`.
2. Enable **Developer mode** (top-right corner).
3. Click **Load unpacked** and select the `dist` folder generated by the build.

### 4. Configuration
1. Click the CredLens AI extension icon in your toolbar.
2. Enter your **Gemini API Key** and click **Save**.

## Usage
- Navigate to [YouTube Shorts](https://www.youtube.com/shorts/).
- Watch a video for at least 3 seconds.
- If factual claims are detected, a credibility indicator will appear in the top-right corner.
- If the content is detected as satire or comedy, no indicator will be shown.

## Project Structure
- `src/content/`: Main logic for YouTube Shorts interaction.
- `src/services/`: Gemini API integration.
- `src/components/`: React UI components (Overlay).
- `src/App.tsx`: Popup UI for settings.
- `manifest.json`: Extension configuration.

## Development
To run in development mode with HMR (Note: Extension HMR is limited):
```bash
npm run dev
```

To build for production:
```bash
npm run build
```
